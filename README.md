## Description

Install UMD4 repositories.

## Setup

This module is available on puppet forge:

```
puppet module install cnafsd-umd4
```

## Usage

Use this module as follow:

```
include umd4
```

## Limitations

It works only on RedHat CentOS 7 distributions.
